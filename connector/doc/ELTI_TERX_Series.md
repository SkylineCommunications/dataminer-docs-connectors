# ELTI TERX Series

## About

The ELTI TERX Series monitors an array of ELTI TERX transmitters designed for digital TV networks.

## Key Features

- **Changeover Configuration and Monitoring**: Set up and monitor changeover capabilities of the agent, such as the states of the multiple available devices, multiple mode configurations and status such as temperature and pressure.

- **Agent Alarm Monitoring**: Monitor the safety of the agent with data from the power supply, input status and general failures.
