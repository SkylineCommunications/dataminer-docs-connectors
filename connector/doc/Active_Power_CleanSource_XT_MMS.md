# Active Power CleanSource XT MMS

## About

The Active Power CleanSource XT MMS connector enables DataMiner integration and monitoring of Active Power's CleanSource XT Multi-Module System (MMS). Through SNMP, this connector retrieves operational and performance data from the device and all added modules.

## Key Features

- **Remote monitoring**: Monitor the Active Power CleanSource XT MMS remotely via DataMiner.
- **Trending**: Configure trending to keep track of the trends in the retrieved data.
- **Alarming**: Get notified of any issues or potential issues through proactive monitoring and issue detection.

## Use Case

**Challenge**: Operators need to ensure continuous uptime by monitoring multiple Active Power CleanSource XT MMS devices across various locations.

**Solution**: Use the Active Power CleanSource XT MMS connector to integrate these devices into DataMiner, enabling real-time off-site monitoring of all operational parameters and alarms.

**Benefit**: Early detection of potential issues and proactive maintenance, leading to improved uptime and reliability of the power systems.
