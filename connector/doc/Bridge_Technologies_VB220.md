# Bridge Technologies VB220

> [!IMPORTANT]
> This connector is no longer supported by Skyline.
> - Is your probe hardware version 4 or higher? -> Use [Bridge Technologies VB Probe Series](https://catalog.dataminer.services/details/f1fa63a6-62b8-4817-82b6-7f37efb7371f).
> - Is your probe hardware version 3 or lower? -> You can still use this Bridge Technologies VB220 connector but is provided as is. We will no longer provide new features nor fixes for it knowing that those very old hardware versions are no longer supported by the vendor either.
