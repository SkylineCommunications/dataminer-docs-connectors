# Bridge Technologies VB220

> [!IMPORTANT]
> 
> This connector is no longer supported by Skyline.
>
> - Does your probe use hardware **version 4 or higher**? Use [Bridge Technologies VB Probe Series](https://catalog.dataminer.services/details/f1fa63a6-62b8-4817-82b6-7f37efb7371f).
> - Does your probe use hardware **version 3 or lower**? You can still use this **Bridge Technologies VB220** connector, but note that it is provided as is. No more new features or fixes will be provided for it, as these very old hardware versions are no longer supported by the vendor either.
