# Smappee Platform

## About

The Smappee Platform connector enables centralized monitoring of Smappee devices by integrating with the Smappee API. It provides users with access to real-time charging data directly within DataMiner.

## Key Features

- **Charging Station**: Retrieve and monitor real-time Charging information.
- **Charging Session Data**: Access analysis and monitoring data for all charging sessions for the charging stations.
- **Cost Analysis data**: Monitor the cost of each location.
