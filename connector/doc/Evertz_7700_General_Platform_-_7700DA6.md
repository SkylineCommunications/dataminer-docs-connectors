# Evertz 7700 General Platform - 7700DA6

> [!NOTE]
> This connector is deprecated. Please use [Evertz 7x00 General Platform](xref:Connector_help_Evertz_7x00_General_Platform) instead.

## About

The **7700DA6** is a 1x6 SD-SDI distribution amplifier. The 7700DA6 series features an auto-equalized input with six reclocked outputs. Supported types: 7700DA6-HD-L or 7700DA6-L.

An **SNMP** connection is used in order to successfully retrieve and configure the device's information. There are also different possibilities for **alarm monitoring** and **trending**.

This connector is automatically generated by connector **Evertz 7700 General Platform**.
