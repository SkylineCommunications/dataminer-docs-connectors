# Evertz 7x00 General Platform - 7707VT4HS

## About

Single card TDM multiplexer for two HD-SDI signals, or one HD-SDI signal and three SDI/DVB-ASI signals, or four SDI/DVB-ASI signals.

An **SNMP** connection is used in order to successfully retrieve and configure the device's information. There are also different possibilities available for **alarm monitoring** and **trending**.

This connector is automatically generated by the connector **Evertz 7x00 General Platform**.

## Configuration

This connector is used by DVEs that are **automatically created** by the parent element. No user input is required.

## How to Use

The DVE consists of the following data pages:

- **Monitor**: Contains general information: **Card Type**, **Card Status, Video Monitoring**, and **Misc Monitoring**.
- **Control**: Contains controls for **Cable Length Threshold** and **Misc** parameters.
- **Traps**: Contains controls for **Traps Configuration** and **Alarm Status**.
