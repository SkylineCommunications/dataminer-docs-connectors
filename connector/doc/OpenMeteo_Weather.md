# Open-Meteo Weather

## About

The Open-Meteo Weather connector enables centralized monitoring of weather conditions by integrating with the Open-Meteo API. It provides users with access to real-time weather data and forecasts for configured locations directly within DataMiner.

## Key Features

- **Current weather monitoring**: Retrieve and monitor real-time weather conditions for configured locations.
- **Quarter-hourly forecast data**: Access weather forecasts at 15-minute intervals for short-term analysis.
- **Hourly forecast data**: Monitor hourly weather forecasts to support long-term operational planning.
