# Harmonic NSG9000 40G

## About

The Harmonic NSG 9000-40G is a high-density universal edgeQAM (EQAM) system designed to serve as a digital video gateway for cable operators. It multiplexes on-demand content over IP networks and is capable of reaching up to 648 QAM-RF output transport streams.

The **Harmonic NSG9000 40G** is an HTTP connector designed to provide comprehensive control and monitoring for the Harmonic NSG9000 40G EdgeQAM device.

## Key Features

- **Hierarchical overview**: Provides an overview of all device parts with a tree view, allowing you to sort and view QAMs by RF and module.

- **VOD management**: Provides dedicated controls for Video on Demand (VOD) settings, including session inactivity teardown thresholds, PID remapping methods (start/end ranges), and PAT/PMT intervals.

- **System health monitoring and control**: Lists all active alarms, and allows you to set limits for concurrent alarms and rate timeframes for alarm storm prevention.

- **Redundancy and EdgeCluster**: Allows configuration of EdgeCluster settings, enabling you to switch device modes (Standalone, Standalone + Mute all RFs, or EdgeCluster) and identify primary versus backup peer devices.

- **Module and QAM status monitoring**: Provides real-time status information for all modules and their associated QAM transport streams, including bitrates and services.

- **CAS and encryption**: Provides features to manage DVB (Session-based and Tier-based) and Privacy Mode settings, alongside an ECM Group Table for managing encryption groups.
