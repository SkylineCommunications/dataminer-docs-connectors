# Meinberg Lantime M300

> [!IMPORTANT]
>
> This connector is no longer supported by Skyline.
>
> - Does your Lantime run firmware **version 7.06.xxx or higher**? Use the [Meinberg LANTIME Non-Modular API V10](https://catalog.dataminer.services/details/761ea7e0-61b0-45ec-a952-adb798679b26) connector.
> - Does your Lantime run firmware **version 7.04.xxx**? Use the [Meinberg LANTIME Non-Modular](https://catalog.dataminer.services/details/1cd29c9c-7e42-45a8-9ae9-4309842c3dae) connector.
