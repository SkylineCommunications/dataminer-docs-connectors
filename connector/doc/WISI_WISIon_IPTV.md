# WISI WISIon IPTV

## About

The WISI WISIon IPTV is designed for continuous monitoring. With this connector, you can monitor the WISI WISIon IPTV's data with DataMiner's alarm monitoring and trending features.

## Key Features

- **Core configuration**: Monitor the settings of your device.
- **Web interface**: Check the whole device and its configurations directly from the element.

## Technical Info

### Prerequisites

- **Credentials**: To use the **web interface** feature, you will need a username and password for the device.
