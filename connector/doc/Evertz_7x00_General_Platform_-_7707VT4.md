# Evertz 7x00 General Platform - 7707VT4

## About

This connector is used to monitor and control a single card TDM multiplexer for four synchronous or asynchronous 270 Mb/s SDI, SDTi or DVB-ASI video signals.

An **SNMP** connection is used in order to successfully retrieve and configure the device's information. There are also different possibilities available for **alarm monitoring** and **trending**.

This connector is automatically generated by the connector **Evertz 7x00 General Platform**.

## Configuration

This connector is used by DVEs that are **automatically created** by the parent element. No user input is required.

## How to Use

The DVE consists of the following data pages:

- **General**: contains general information, such as the **Local Remote Jumper** and **Password** configuration.
- **Control**: Allows you to monitor and control **Video** and **Cable**.
- **Alarms**: Contains controls for **Traps Configuration** and **Alarm Status**.
