# Nokia_Altiplano_REST_Interface

## About

The **Nokia Altiplano REST Interface** is an HTTP driver designed to retrieve API data from designated endpoints on the OLT (Optical Line Terminal) side. It serves as a key component in facilitating communication between the Altiplano platform and network devices.

## Key Features

- **OLTs Data**: Collects and returns OLT-specific data in response to requests initiated via InterApp messages.
- **InterApp Message Integration**: Supports the reception and execution of commands delivered through InterApp messaging, enabling API data requests and responses to the master element.
- **API Request Inventory**: Maintains a comprehensive log of executed API requests, including both inbound and outbound message records for operational transparency.
