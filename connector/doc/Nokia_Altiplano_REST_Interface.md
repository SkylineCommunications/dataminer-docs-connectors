# Nokia_Altiplano_REST_Interface

## About

The Nokia_Altiplano_REST_Interface is an HTTP driver which returns API data from the specific endpoint (OLTs side).

## Key Features

- **OLTs Data**: Retrieve all OLT data that are requested via InterApp messages.
- **InterApp Messages**: Implement InterApp messages to receive commands to request API data and return it to its master element.
- **Inventory**: Show the inventory of the API requests executed (includes inbound/outbound messages).
