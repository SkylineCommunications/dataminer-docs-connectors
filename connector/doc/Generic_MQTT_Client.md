# Generic MQTT Client

## About

The Generic MQTT Client connector provides DataMiner operators with an MQTT communication solution. It enables DataMiner to connect to an MQTT broker and offers real-time visibility into broker activity through general connection statistics and status information.

In addition to monitoring the MQTT connection, the connector supports subscribing to user-defined topics to receive messages in real time and publishing messages to specified topics. This enables bidirectional communication with MQTT-based systems.

## Key Features

- **Broker monitoring**: Provides an overview of the MQTT broker connection status and key statistics, giving operators clear insight into the activity of the connection and the broker.
- **Topic subscriptions**: Subscribes to one or more user-defined topics, allowing DataMiner to receive and display incoming messages in real time.
- **Message publishing**: Sends messages to specified topics directly from DataMiner, enabling bidirectional communication with MQTT-based systems.
