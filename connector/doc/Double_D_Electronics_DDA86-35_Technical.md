# Double D Electronics DDA86-35

## About

The connector interacts with the Double D Electronics DDA86‑35 device and allows the end user to view device information, monitor alarms, and control the lock and position of the available switches.

## Configuration

### Connections

This connector uses a serial connection and requires the following input during element creation:

SERIAL CONNECTION:

- Interface connection:

  - **IP address/host**: The polling IP of the device.
  - **IP port**: The IP port of the device.
  - **Bus address**: The bus address of the device.

## How to Use

### General Page

The General page provides key device information such as the unit type, software version, site‑specific details, and the highest supported switch or HPA index. It also includes parameters for alarm status, PSU faults, chain summary alarms, and HPA‑related alarms.

In addition, you can view or adjust control‑related settings including control mode, global device mode, mute override, and remote mode lockout.

### Switch Page

The Switch page offers direct control and monitoring of each available switch.

For every switch, you can view its status, check whether it is locked, and read or set the current switch position.
