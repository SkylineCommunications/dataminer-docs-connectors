# Evertz 7700 General Platform - 7700DA7

> [!NOTE]
> This connector is deprecated. Please use [Evertz 7x00 General Platform](xref:Connector_help_Evertz_7x00_General_Platform) instead.

## About

The **7700DA7** series distribution amplifier features an auto-equalized input with seven reclocked outputs. Supported types: 7700DA7-HD or 7700DA7.

An **SNMP** connection is used in order to successfully retrieve and configure the device's information. There are different possibilities for **alarming monitoring** and **trending**.

This connector is automatically generated by connector **Evertz 7700 General Platform**.
