# Hikvision DS-9664NI-M8 NVR

## About

The Hikvision DS-9664NI-M8 NVR connector provides standardized access to the recorder's live and recorded video and device telemetry. The connector exposes health and status metrics and diagnostics for integration with video management and analytics platforms.

## Key Features

- **Standardized live and recorded video access**: Programmatic access to live streams and onboard recordings.
- **Device telemetry and configuration**: Exposes device and channel telemetry (status, firmware, storage, uptime) and provides configuration APIs for channel mapping and stream parameter adjustments.

## Technical Reference

### Prerequisites

- **User credentials** are required for authentication of the HTTP calls.
