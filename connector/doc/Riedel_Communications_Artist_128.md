# Riedel Communications Artist 128

> [!IMPORTANT]
>
> Skyline Communications no longer supports this connector.
>
> - Does your device use firmware version **RRCS 8.5 or higher**? Use [Riedel Communications ARTIST](https://catalog.dataminer.services/details/991cd738-f706-4e58-b5f7-9f1f16dee733).
> - Does your device use firmware version **RRCS 8.4 or lower**? You can still use this **Riedel Communications Artist 128** connector, but note that it is provided as is. No more new features or fixes will be provided for it, as these old firmware versions are no longer supported.
