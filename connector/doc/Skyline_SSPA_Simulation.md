# Skyline SSPA Simulation

## About

The Skyline SSPA Simulation connector emulates an SSPA device for demo, testing, and training purposes. It provides a virtual environment where users can monitor and adjust key amplifier parameters without needing physical hardware.

## Key Features

- **Ideal for demos and training**: Show system behavior and teach workflows safely.  
- **Safe testing**: Experiment with parameter changes without impacting real equipment.
