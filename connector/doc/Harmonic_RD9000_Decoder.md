# Harmonic RD9000 Decoder

## About

This is a DataMiner connector for the **Harmonic RD9000 Decoder**, a multiformat professional **single or multi-slice UHD decoder that is HDR ready**.

### Key Features

- **Alarm monitoring of the devices**: Every alarm triggered from the devices can be monitored in real time.
- **Configuration**: Configure the decoder settings to satisfy operational requirements.
