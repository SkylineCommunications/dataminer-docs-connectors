# FS 3900-R Series

## About

The FS S3900-R Series is a 10GE switch with 24 or 48 1G ports. This connector listens for interface data from the device, and reads the status and errors of the device.

## Key Features

- **Detailed information for monitoring**: The connector displays detailed information about the switch interfaces, NTP, DHCP server, IP addresses, VLAN, and STP, allowing real-time monitoring in DataMiner.
