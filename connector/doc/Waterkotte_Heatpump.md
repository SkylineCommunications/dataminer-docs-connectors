# Waterkotte Heatpump

## About

The Waterkotte Heatpump connector enables monitoring of the heatpump of Waterkotte devices via modbus. It provides users with access to real-time data directly within DataMiner.

## Key Features

- **Compressors**: Retrieve and monitor real-time compressor information.
- **Alround info heatpump**: Access analysis and monitoring data for all .
- **Slave unit**: Retrieve data from the slave unit.
