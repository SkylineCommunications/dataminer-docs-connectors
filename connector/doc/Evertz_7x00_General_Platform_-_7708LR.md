# Evertz 7x00 General Platform - 7708LR

## About

The **7708LR** cards accept a fiber-optic input from a companion transmitter and provide an electrical output signal. Supported types: 7708LRA and 7708LR-H.

An **SNMP** connection is used in order to successfully retrieve and configure the device's information. There are different possibilities for **alarming monitoring** and **trending**.

This connector is automatically generated by connector **Evertz 7x00 General Platform.**

## Configuration

This connector is used by DVEs that are automatically created by the parent element. No user input is required.
