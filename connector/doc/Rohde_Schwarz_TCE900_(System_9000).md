# Rohde Schwarz TCE900 (System 9000)

## About

The Rohde Schwarz TCE900 is a transmitter controller from the System 9000 generation designed for both analog and digital TV transmissions.

In this connector the user can take advantage of the versatility of the TCE900 by monitoring multiple types of transmitters, inputs and even the rack unit. 

## Key Features

- **Transmitter Monitoring and Control**: Monitor and control multiple transmitter configurations besides the main, such as multi, N+1 and backup.

- **Input and Output Overview**: Provides an overview of multiple input types, such as ATV, DAB, FM and ATSC30. The DVBT, DVBT2, ISDBT, DTM and ATSC30 outputs are also present.

- **Hardware Monitoring**: Monitor the TCE900 rack unit with information about the sensors present, amplifiers, exciter and cooling system.
