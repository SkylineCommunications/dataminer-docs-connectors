# Paneda DAB Head-End

## About

The Paneda DAB Head-End connector provides a centralized way to monitor the Paneda DAB/DAB+ broadcast system within DataMiner. It retrieves and combines data from multiple providers configured in the system, giving users a comprehensive overview of the overall system status as well as specific insights for each provider. By integrating with both SNMP and HTTPS, this connector ensures real-time access to critical information across devices, audio services, and data services for each provider.

## Key Features

- **Multi-provider monitoring**: Discovers and retrieves data for the providers configured in the system, allowing you to monitor multiple providers from a single element.
- **Comprehensive service visibility**: Retrieves status information across devices, audio services, and data services for every provider in the system.
- **System overview**: Displays general system-level data shared across all providers separately from provider-specific data.
