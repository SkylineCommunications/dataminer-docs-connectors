# Rohde & Schwarz NetCCU800

## About

This is a DataMiner connector for the **Rohde & Schwarz NetCCU800** control unit.
This is commonly used for FM and TV transmitters.

### Key Features

- **Dual Device Type Support**: Switch between DVB and ISDBT device configurations for flexible deployment scenarios.
- **Comprehensive Module Monitoring**: Monitor and control DVB receivers, transmitters, exciters (A & B), amplifiers, and pump units.
- **Event & Alarm Management**: Real-time SNMP trap reception with configurable event masks, priority levels, and fault tracking across GPS, NSU, and pump unit events.
- **Transmitter Control**: Remote restart capabilities, fault reset functionality, and transmitter command execution.
- **SFN Delay Calculation**: Automatic calculation of synchronous frequency network (SFN) delay parameters between Exciters A and B.
- **Amplifier Management**: Status monitoring, alarm tracking, and rack-level information for system amplifiers including Sx801 models.
- **Network Time Synchronization**: Built-in NTP server configuration for accurate time management.
- **RF Probe Monitoring**: Track and configure RF probe parameters for signal quality assessment.
- **Software Update Management**: Remote software update configuration and deployment capabilities.
- **Trap Sink Configuration**: Customizable SNMP trap destination management for integration with monitoring systems.

## Prerequisites

- The Get/Set community strings are required for reading and setting values onto the device.
