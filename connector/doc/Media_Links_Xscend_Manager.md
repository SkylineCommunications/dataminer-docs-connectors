# Media Links Xscend Manager

## About

Media Links Xscend is a media platform used to transport video/data services.

The *Media Links Xscend Manager* connector is used to monitor Xscend chassis and the installed modules.

## Key Features

- **Alarm Monitoring**: The connector provides the option to monitor all alarms (including inactive alarms) or only active alarms.
- **Metrics Monitoring**: In addition to the alarm monitoring, it is also possible to monitor basic metrics for each individual module.
