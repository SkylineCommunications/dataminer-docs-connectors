# Specialty Microwave Corp 8682-500

## About

The **Specialty Microwave Corp 8682-500** will monitor and configure a Specialty Microwave Corp 8682-500 1:4 protection switch.

## Configuration

### Connections

This connector uses a serial connection and requires the following input during element creation:

SERIAL CONNECTION:

- **IP address/host**: The polling IP of the device, e.g. *10.11.12.13.*
- **IP port**: The port of the connected device.
- **Bus address**: The bus address of the connected device's relay processor, ranging from *33* to *126* inclusive.

## How to Use

On the **General** page, you can find the following information about the device:

- Position of switches S1 through S4.
- Position of switches C1 through C4.
- Fail indicators for HPA 1 through 4, plus the standby HPA.
- Fail indicator for the Sum Power.
- Automatic/Manual mode indicator.

On this page, you can also toggle between automatic and manual mode, and toggle the position of switches 1 through 4.
