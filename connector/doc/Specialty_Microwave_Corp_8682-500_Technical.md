# Specialty Microwave Corp 8682-500

With this connector it is possible to monitor a **Specialty Microwave Corp 8682-500** serial device.

## About

The **Specialty Microwave Corp 8682-500** will monitor and configure a Specialty Microwave Corp 8682-500 1:4 Protection switch.

## Installation and configuration

### Creation

The **Specialty Microwave Corp 8682-500** is a serial connector. The IP address, Port, and Bus address need to be configured during creation of the element.

**SERIAL CONNECTION**:

- **IP address/host**: The polling IP of the device, e.g. *10.11.12.13.*
- **IP port**: The port of the connected device.
- **Bus address**: The bus address of the connected deviceâ€™s relay processor, ranging from *33* to *126* inclusive.

## Usage

### General Page

This page displays general information about the device:

- Position of switches S1 through S4.
- Position of switches C1 through C4.
- Fail indicators for HPA 1 through 4, plus the standby HPA.
- Fail indicator for the Sum Power.
- Automatic/Manual mode indicator.

The page also allows the user to toggle between automatic and manual mode, and to toggle the position of switches 1 through 4.
