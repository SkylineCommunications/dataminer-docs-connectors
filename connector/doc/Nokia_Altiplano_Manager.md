# Nokia Altiplano Manager

## About

The **Nokia Altiplano Manager** connector enables virtual integration between network elements in the **Altiplano Solution**. Operating in either Master or Slave mode, it supports REST API and Kafka-based communication, inventory management, and InterApp messaging. Configuration is streamlined through a virtual connection, with role-specific capabilities for data exchange, monitoring, and operational control

This connector ensures efficient communication, coordination, and execution across the network by acting as a manager/broadcast element.

## Key Features

- **Unified Request Handling**: Supports both Kafka and HTTP-based API requests.
- **Broadcast Distribution**: Automatically forwards requests to all connected drivers for parallel execution.
- **Virtualized Operation**: Operates as a virtual entity within the Altiplano solution.
- **Scalable Coordination**: Enables scalable orchestration across multiple drivers and network elements.
