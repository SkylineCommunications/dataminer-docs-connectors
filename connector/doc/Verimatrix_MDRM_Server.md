# Verimatrix MDRM Server

## About

The Verimatrix MDRM connector gives a single overview of the MDRM server.
There is also the possiblity to do some basic configurations.

## Key Features

- **24/7 monitoring**: Constant monitoring to support continuous status updates.
- **Process overview**: Have an overview of all running processes on the MDRM server and being able to fix an issue through a command.
