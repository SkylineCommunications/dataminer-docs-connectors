# Verimatrix MDRM Server

## About

The Verimatrix MDRM connector gives a single overview of the MDRM server and allows users to configure basic settings.

## Key Features

- **24/7 monitoring**: Constant monitoring to support continuous status updates.
- **Process overview**: Complete overview of the processes running on the MDRM server, with the ability to fix an issue through a command.
