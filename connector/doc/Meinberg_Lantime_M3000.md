# Meinberg Lantime M3000

> [!IMPORTANT]
>
> This connector is no longer supported by Skyline.
>
> - Does your Lantime run firmware **version 7.06.xxx or higher**? Use the [Meinberg LANTIME Modular API V10](https://catalog.dataminer.services/details/ea6828f3-6b90-4884-beff-ab993c09fe0e) connector.
> - Does your Lantime run firmware **version 7.04.xxx**? Use the [Meinberg LANTIME Modular](https://catalog.dataminer.services/details/078c8342-bcbe-4fbb-b1cd-166abcbe4658) connector.
