# International Datacasting SFX3100

## About

The International Datacasting SFX3100 is designed for continuous monitoring. With this connector, you can monitor the International Datacasting SFX3100's data with DataMiner's alarm monitoring and trending features.

## Key Features

- **Core configuration**: Monitor and change the settings of your device.
- **Trap support**: Make sure your element is always up to date through traps received directly in DataMiner.
